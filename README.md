## 🌟 Key Features

- **3D Interactive Experience:** Built with **React Three Fiber (R3F)** and **Three.js**, featuring a smooth scroll-controlled "Quantum Knot" that dynamically changes colors and shape.
- **Dual Language Support:** Fully localized in **Turkish** and **English** with a real-time toggle.
- **Modern UI/UX:** Minimalist dark-themed design using **Space Grotesk** and **Playfair Display** typography.
- **Dynamic Typewriter Effect:** Showcasing multiple roles and expertise fields.
- **Fully Responsive:** Optimized for all screen sizes with a focus on high-performance rendering.

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **3D Library:** React Three Fiber, @react-three/drei
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion & Three.js Clock
- **Typography:** Google Fonts Integration
